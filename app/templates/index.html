    About Kanseki Repository

    This website provides access to the texts in the Kanseki Repository. At the moment, there is only very simple search functionality, but it is planned to extend this depending on the needs of the users.

    More information about the Kanseki Repository, its design considerations and aims can be found here: Kanripo Blog.

    A more sophisticated and powerful way to access the repository is through Mandoku, an extension of Org-Mode for Emacs.
